홈 화면에 saudade 를 추가하세요.

앱스토어 거치지 않음 · 자동 업데이트 · 오프라인 작동

iOS — Safari

  1. Safari 로 saudade.pages.dev 접속
  2. 하단 가운데 공유 버튼 (위로 화살표 사각형)
  3. 아래로 스크롤 → "홈 화면에 추가"
  4. 이름 확인 후 우측 상단 "추가"

iOS 16.4 이상이면 푸시 알림도 받을 수 있습니다 (앱 안에서 동의).

Android — Chrome

  1. Chrome 으로 saudade.pages.dev 접속
  2. 주소창 우측 점 세 개 메뉴
  3. "앱 설치" 또는 "홈 화면에 추가"
  4. 확인 → 앱 아이콘 생성

Desktop — Chrome / Edge

  1. saudade.pages.dev 접속
  2. 주소창 우측에 작은 설치 아이콘 (모니터 + 화살표)
  3. 클릭 → 독립 창으로 실행

PWA 라 앱스토어 검색 결과엔 안 나옵니다. 친구에게 공유할 땐 saudade.pages.dev 링크를 그대로 보내면 됩니다.

Add saudade to your home screen.

No app store · auto-updates · works offline

iOS — Safari

  1. Open saudade.pages.dev in Safari
  2. Tap the Share button (square + up arrow) at the bottom
  3. Scroll down → "Add to Home Screen"
  4. Confirm name → "Add" in the top right

iOS 16.4+ supports push notifications (opt in inside the app).

Android — Chrome

  1. Open saudade.pages.dev in Chrome
  2. Three-dot menu, top right of the address bar
  3. "Install app" or "Add to Home Screen"
  4. Confirm → app icon appears

Desktop — Chrome / Edge

  1. Open saudade.pages.dev
  2. Look for a small install icon on the right side of the address bar
  3. Click → opens in its own window

It's a PWA, not an app-store listing. To share, just send the saudade.pages.dev URL.

saudade をホーム画面へ。

App Store 不要 · 自動更新 · オフライン対応

iOS — Safari

  1. Safari で saudade.pages.dev を開く
  2. 画面下中央の共有ボタン
  3. 下にスクロール → 「ホーム画面に追加」
  4. 名前を確認して右上「追加」

iOS 16.4 以上ならプッシュ通知も受信可（アプリ内で許可）。

Android — Chrome

  1. Chrome で saudade.pages.dev
  2. アドレスバー右の三点メニュー
  3. 「アプリをインストール」
